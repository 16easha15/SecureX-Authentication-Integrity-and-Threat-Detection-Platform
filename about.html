About SecureX

Project Overview

SecureX is a cybersecurity-based authentication system project designed to demonstrate modern login security mechanisms.


Features included in this project:

  • SHA-256 Password Hashing
  • Register / Login System
  • Role-Based Access Control (Admin/User)
  • MFA Authentication
  • Login History Tracking
  • Suspicious Login Detection
  • 2-Minute Account Lock System
  • Integrity Checker
  • Security Score Meter
  • Admin Monitoring Panel
  • Responsive Glassmorphism UI

This project is built using HTML, CSS, JavaScript, Node.js, Express.js, and MySQL.
